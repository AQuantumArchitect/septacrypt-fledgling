# The physical SeptaCrypt — peripheralarbor.com/gallery/Projects/SeptaCrypt/ (digest)

Archived 2026-07-16 via web fetch (AI-digested).

A seven-faced physical cryptographic object with spheres mounted loose in
sockets ("a dull clicking sound when the Septa-Crypt is manipulated");
six-sided peen marks on the nameplate capture the spheres.

- **Three primary faces** — encoded messages in an invented language;
  "no one has ever figured out the message encoded on the three faces."
- **Three tertiary faces** — a dedication, the Tryop logo, "a Final Face."
- **Bottom face** — "the key for decoding" the primary messages.

This is the arrangement FLEDGELING_SEPTACRYPT_PLAN.md §2 maps onto software:
three primary world-model faces (Architect/Structure, SpaceWheat/Dynamics,
umwelt/Belief), three tertiary process faces (Berry Tape/Kairos, Knot
Ledger/History, Fledgeling/Agency), bottom key face (Spirit Cube/Meaning).
A decryption guide exists on the maker's blog (not archived).
